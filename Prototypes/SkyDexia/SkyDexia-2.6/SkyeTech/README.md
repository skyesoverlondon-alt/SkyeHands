# SkyeTech Module Hub

SkyeTech is the modular import lane for this workspace. It is designed so inexperienced builders can start from guided building blocks instead of a blank slate.

## How To Use

1. Keep reusable modules and starter blueprints in this folder.
2. Add one module at a time and document what it does.
3. Ask SkyDexia to wire selected modules into the active app scaffold.
